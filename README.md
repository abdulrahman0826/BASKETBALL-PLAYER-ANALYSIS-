# BASKETBALL-PLAYER-ANALYSIS-
Basketball player statistical dashboard using power bi desktop
üèÄ Basketball Player Statistics Dashboard (Power BI)
This repository contains a dynamic Power BI dashboard (Basketball_Dashboard.pbix) designed to visualize and analyze key performance indicators (KPIs) for basketball players across different seasons and teams.

üí° Overview
The dashboard provides an interactive and visually engaging way to explore player and team statistics, allowing users to quickly identify top performers, analyze efficiency metrics, and compare data across seasons.

üíæ Data Source
The dashboard is built upon the following raw data file, which is also included in this repository:
Basketball.csv: This is the primary dataset containing detailed individual player statistics. Key metrics include:
Scoring: Points (PTS), Field Goals Made (FGM), Field Goal Attempts (FGA), Free Throw Percentage (FT_PCT).
Playmaking/Rebounding: Assists (AST), Rebounds (REB), Offensive and Defensive Rebounds (OREB, DREB).
Efficiency: Minutes Played (MIN), Steals (STL), Blocks (BLK), Turnovers (Turnover).

‚ú® Key Dashboard Features
The Basketball_Dashboard.pbix file offers the following interactive components and visualizations:
Top Performer Charts: Visualizations dedicated to highlighting leaders in primary statistics (e.g., top scorers, rebound leaders).
Efficiency Metrics: Calculated columns and measures (using DAX) to track advanced metrics, such as Assist-to-Turnover Ratio or True Shooting Percentage.
Interactive Slicers: Users can filter the entire dashboard instantly by Year and Team, enabling focused analysis on specific seasons or franchises.
Data Transformation: The data was cleaned and modeled using Power Query to ensure data integrity and optimal performance within the Power BI data model.
